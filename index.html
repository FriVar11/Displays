Display: Lo que mostramos en nuestra pantalla

Display None

Este tipo de display es especial porque nos permite dejar de visualizar contenido dentro de mi sitio.

Cuando ocupamos un display none, las etiquetas dejan de verse pero sigue el código ahí.

Display Block

El elemento genera un cuadro de elemento de bloque

Significa que mi bloque (recuerda que todo elemento en html es un bloque), ocupa todo el espacio disponible de izquierda a derecha, o dicho de otra forma, todo el renglón.

Display Inline

Significa que el contenedor o bloque se adapta al ancho de su contenido hijo.

Display Inline-Block

Si tú le pones tamaño a un display block, funciona, pero sigue ocupando todo el renglón

Si tú le pones tamaño a un displayInLine, ignora las propiedades de alto y ancho (height and width).

Esto puede ser problemático si yo quiero controlar el tamaño de mi componente y que el contenido esté en la misma línea, para eso existe el displayInLineBlock
